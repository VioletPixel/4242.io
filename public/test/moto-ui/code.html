This form makes a direct request to the Stripe API using a publishable key, without Stripe.js, to create a Payment Method using the card information provided. The resulting Payment Method can then be used server-side for a MOTO payment.

Note: This approach is only appropriate for entities that are PCI-compliant, where agents are handling raw card details to facilitate MOTO payments.

Stripe.js, Elements, or a Stripe-hosted UI (like Checkout) are only suitable for use by cardholders and should not be used for MOTO payments.
